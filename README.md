近年来，西南交通大学计算机与人工智能学院国家云服务科技团队长期承担国家重点研发计划、国家科技支撑计划等重大科研项目，并深度参与相关指南论证、项目申报、组织实施、成果评价及验收结题等工作。有关公开材料显示，团队负责人长期担任国家科技计划相关专家组负责人或评审专家，而其本人及关联团队同时作为项目承担单位、成果完成人和受益主体，相关项目在指南形成、项目立项、组织实施、成果验收等多个关键环节可能存在利益关联和回避机制落实不到位的问题，涉嫌形成"既当裁判员、又当运动员"的利益闭环，影响国家科技计划管理的独立性、公正性和严肃性。

经对多个国家重点研发计划项目公开材料进行梳理和比对，发现相关项目在平台应用规模、业务协同效果、成果证明、应用效益、标准应用、论文支撑等方面存在大量值得进一步核查的情况。例如，部分项目所宣称的第三方云平台应用规模、协作企业数量、业务协同效率等指标，与平台实际运行情况可能存在明显差异；部分应用证明、效益分析报告及标准应用证明存在数据来源不清、统计口径模糊、缺乏独立审计和第三方核验等问题；部分所谓"多链联动""跨链互动""超链融动"等核心成果，实际可能主要来源于历史业务数据汇总或企业既有业务，并不能充分证明国家重点研发计划成果已实现相应技术目标。

同时，项目成果形成过程中，还可能存在学生长期承担项目申报、系统开发、报告编写、软件著作权申请、验收材料整理、应用证明收集等大量项目性工作，与研究生培养目标偏离；若进一步查实存在无正当理由拖延学生毕业、成果署名与实际贡献不符、项目申报和验收材料失实等情况，则不仅涉及研究生合法权益保护问题，更可能涉及国家科技计划项目管理、财政科研资金使用以及科技诚信等方面的违法违规风险。

此外，据公开履历及项目材料反映，部分教师依托上述国家重大科研项目，在较短时间内完成职称晋升、人才计划申报及荣誉获得。有关成果是否真实反映个人学术贡献，是否存在依托项目成果不当获取学术评价和职务晋升等情况，也有必要结合项目原始资料进行核查。

鉴于相关项目涉及国家重点研发计划及配套省部级项目，财政资金规模较大，相关人员长期担任国家科技计划专家，且部分学校管理人员与项目之间存在一定关联关系，为确保调查工作的客观、公正和独立，恳请有关部门依法成立独立调查组，对项目指南编制、专家评审、项目申报、经费使用、成果形成、论文及专利署名、平台运行数据、应用证明、验收材料以及职称晋升等全过程开展穿透式调查，以原始系统数据、服务器日志、财务凭证、项目档案及独立访谈等客观证据为依据，全面查明是否存在科研诚信、科技计划管理及利益冲突等方面的违法违规问题，并依法依规追究相关责任。
